# Style Guide: ClearDesk

## Design System

- **Background:** Slate-950 (Dark Mode Primary)
- **Cards:** Slate-900 with thin border (Slate-800)
- **Typography:** Sans-serif (Inter or System UI)
- **Accent:** Emerald-500 (for success/completed) or Indigo-500 (for "Promote" actions)

## Component Rules

- Use **Custom SVG Components** (located in `app/components/Icons`) for consistency when no https://icones.js.org/collection/heroicons icon is available
- Buttons should have a subtle `transition-all` on hover.
- Use `shadcn-vue` style primitives where possible.
